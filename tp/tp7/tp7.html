Exercice 1 : Zoning à partir du lien

À partir du lien : http://j-willette.developpez.com/tutoriels/html/les-bases-du-html/?page=page_1
• Utiliser l'inspecteur d'élément pour représenter sous forme de "boîte" le zoning de la page. • Précisez pour chaque zone la balise utilisée. • Pour la partie article, notez l'architecture du site.

Exemple de zoning (dans l'esprit de la page) :

HEADER
<header>
MENU 1
<nav>
RECHERCHE
<form>
LOGO/PUB
<div>
MENU 2
<nav>
ARTICLE
<article>
FOOTER
<footer>

Exercice 2 : Reproduire le zoning de la page ci-dessous

On ne s'intéresse pas au contenu, mais uniquement à la structure (le zoning). Chaque page doit garder ses proportions.
Voici le code HTML donné dans l'énoncé, réécrit avec colorisation. Créez un fichier CSS séparé si souhaité, mais ci-dessous nous vous montrons une version "inline" pour illustrer l'affichage.

<!DOCTYPE html>
<html>
  <head>
    <title>TODO supply a title</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
      .entete { border:1px solid; }
      .menu1 { border:1px solid blue; height:30px; display:inline-block; float:left; margin:1px; }
      .recherche { border:1px solid red; height:30px; float:right; margin:1px; }
      .cadreLogoPub { border:1px solid blueviolet; clear:left; margin:1px; }
      .logo { display:inline-block; border:1px solid chocolate; margin:1px; }
      .emploi { display:inline-block; border:1px solid chartreuse; }
      .main { margin-top:2px; margin-bottom:2px; background-color:cadetblue; height:400px; }
      .pdp { margin-top:2px; margin-bottom:2px; background-color:chartreuse; height:50px; }
    </style>
  </head>
  <body>
    <header class="entete">
      <div>
        <nav class="menu1">ZONE DE MENU 1</nav>
        <form class="recherche">
          <input type="text" />
          <input type="submit" value="rechercher" />
        </form>
      </div>
      <div class="cadreLogoPub">
        <div class="logo"><img src="images/arrow.png" alt="logo" /></div>
        <div class="emploi">CADRE EMPLOI</div>
      </div>
      <nav>ZONE DE MENU 2</nav>
    </header>
    <section class="main">
      <article>ARTICLE</article>
    </section>
    <footer class="pdp">PIED DE PAGE</footer>
  </body>
</html>

Le zoning obtenu (en gardant les mêmes proportions) peut être illustré par :

ENTETE
.entete <header>
MENU 1
.menu1 <nav>
RECHERCHE
.recherche <form>
LOGO / PUB
.cadreLogoPub <div>
MENU 2
<nav>
ARTICLE
.main <section> <article>
PIED DE PAGE
.pdp <footer>
