TP7 - Zoning en CSS

Retrouvez ici les exercices du TP7 sur le zoning des pages web.

Exercice 1 : Zoning de la page

Utilisez l'inspecteur pour représenter le zoning de la page :

  • HEADER : <header>
  • MENU 1 : <nav>
  • RECHERCHE : <form>
  • LOGO / PUB : <div>
  • MENU 2 : <nav>
  • ARTICLE : <article>
  • FOOTER : <footer>

Exercice 2 : Reproduire le zoning sans contenu

Reproduisez le zoning sans contenu réel mais avec la structure HTML appropriée. Exemple :

  • ENTETE : <header>
  • MENU 1 : <nav>
  • RECHERCHE : <form>
  • LOGO / PUB : <div>
  • MENU 2 : <nav>
  • ARTICLE : <article>
  • FOOTER : <footer>

Exercice 3 : Zoning avec une grille CSS

Reproduisez le zoning avec une grille CSS pour organiser les différentes sections de manière responsive.

  • ZONE 1 : <section>
  • ZONE 2 : <section>
  • ZONE 3 : <section>
  • ZONE 4 : <section>

Exercice 4 : Intégration d'images et médias

Ajouter des éléments multimédias (images, vidéos, etc.) dans les différentes zones pour tester l'intégration.

  • IMAGE : <img>
  • VIDÉO : <video>
  • AUDIO : <audio>

Exercice 5 : Flexbox et Alignement

Utilisez flexbox pour aligner les éléments du zoning horizontalement et verticalement.

  • Alignement de l'entête : flex
  • Alignement des zones : justify-content: space-between
